# Sample Chat Data

This directory contains sample conversation data for demonstration purposes.

## Files

- `sample_chat.json`: Example conversation between user and AI companion

## Format

Each chat message contains:
- `role`: The role of the speaker (user, assistant, system)
- `content`: The message content
- `timestamp` (optional): When the message was sent

## Usage

These sample files can be used to:
1. Test the application
2. Demonstrate conversation flows
3. Initialize the app with example data
